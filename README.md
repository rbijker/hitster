# HITSTER — Music Timeline Party Game

A Progressive Web App (PWA) version of the popular HITSTER board game. Optimized for iPhone with Spotify integration.

## Play Now

Open the GitHub Pages URL on your iPhone and add to Home Screen for the full app experience.

## Features

### 4 Game Modes (Official Rules)
| Mode | Rules | Tokens |
|------|-------|--------|
| **Original** | Place card in timeline. Earn tokens by naming title & artist. | 2 |
| **Pro** | Place card AND name title & artist to keep it. | 5 |
| **Expert** | Place card + title + artist + exact year. | 3 |
| **Co-op** | One team. Get 10 cards before tokens run out. | 5 |

### Token System
- **Skip** — 1 token to draw a new song
- **Buy** — 3 tokens to auto-place a card
- **HITSTER!** — Challenge another player's wrong placement and steal the card
- Max 5 tokens per player

### Music Themes
Pop · Rock · Hip-Hop · Electronic · Classics · 80s · 90s · 2000s · 2010s · Dutch Scene

### Spotify Integration
Opens the Spotify app directly via deep linking — no API token or Premium required.

## Install on iPhone

1. Open the GitHub Pages URL in **Safari**
2. Tap **Share** → **Add to Home Screen**
3. App launches fullscreen with its own icon
